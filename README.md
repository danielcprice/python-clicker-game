# python-clicker-game

## Summary
This is the first game that I have made. The idea for this game is that you have a population and resources maybe some world events. Population will increase the amount of resources that you can passively gather. You can spend resources to increase your population and population will eat resources over time. You can research items to get more or different types of resources. If your population drops below 2 you lose. The goal is to grow while maintaining the balance.
